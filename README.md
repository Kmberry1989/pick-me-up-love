# Pick Me Up Love üß∏

A beautifully crafted 3D Claw Machine game that runs right in your browser! Built with Three.js and Cannon.js.

## Features
-   **Realistic Physics:** Prizes tumble, stack, and react to the claw.
-   **Procedural Audio:** Custom synthesizer sounds for arcade ambiance.
-   **Bright Arcade Environment:** Immerse yourself in a pastel-colored wonderland.
-   **Unique Prizes:** Use the claw to collect adorable characters.
-   **Full Cabinet Simulation:** Complete with marquee, joystick, coin slots, and lights.

## How to Play

### Controls
-   **Joystick:** Drag the on-screen joystick to move the claw.
-   **DROP Button:** Press to lower the claw and try your luck!
-   **Coin Slot:** The game starts with credits, but if you run out, just refresh! (Or mod the `credits` variable üòâ)
-   **Collect:** Click the "PRIZE WON" message to claim your reward.

### Objective
Position the claw over a prize, drop it at the perfect moment, and hope for a strong grip!

## Installation & Running

This project is a static web application. You just need a local server to run it.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Kmberry1989/pick-me-up-love.git
    cd pick-me-up-love
    ```

2.  **Install dependencies (optional, for tools):**
    ```bash
    npm install
    ```

3.  **Run the local server:**
    ```bash
    npx http-server .
    ```

4.  **Open in Browser:**
    Navigate to `http://localhost:8080`

## Technologies
-   **Three.js:** 3D Rendering
-   **Cannon.js:** Physics Engine
-   **GLTFLoader:** Model Import
-   **Vanilla JS:** Game Logic
